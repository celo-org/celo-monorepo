## Commands

### Get-metadata

Show information about an address

```
USAGE
  $ celocli identity:get-metadata ADDRESS

ARGUMENTS
  ADDRESS  Address to get metadata for

EXAMPLE
  get-metadata 0x97f7333c51897469E8D98E7af8653aAb468050a3
```

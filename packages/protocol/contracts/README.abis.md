# Celo core contracts ABIS

Smart contracts ABIS for the Celo protocol.

## Usage

### Installation

```bash
npm install @celo/abis@beta
```

or

```bash
yarn add @celo/abis@beta
```

### In your javascript or ts

```js
import Accounts from '@celo/abis/Accounts.json'
```
